## 구현 내용

### 제목

채팅 반복의 한계 : Skill로 절차를 외부화

#### Left — Before: 매 채팅마다 같은 지침을 반복

> 🔧 구현: svg

```text
┌─────────────────────────┐
│  매 대화 시작마다       │
│  ↓                      │
│  [긴 지침 붙여넣기]     │
│  - 체크리스트 N항목     │
│  - 다단계 절차 설명     │
│  - 컨벤션·제약 나열     │
│  ↓                      │
│  토큰 비용 ─ 매번 부과  │
│  실수 위험 ─ 매번 재현  │
└─────────────────────────┘
```

시각화 의도: 매번 반복되는 비효율을 박스 안 무거운 텍스트 더미로 표현

#### Right — After: SKILL.md 한 번 작성, 호출 시에만 로드

> 🔧 구현: svg

```text
┌─────────────────────────┐
│  .claude/skills/         │
│   └─ deploy/             │
│      └─ SKILL.md  ◀─────│
│                          │
│  사용자: "/deploy"       │
│  ↓                       │
│  [SKILL.md 로드 1회]    │
│  ↓                       │
│  토큰 비용 ─ 호출 시만   │
│  버전 관리 ─ git에 커밋  │
└─────────────────────────┘
```

시각화 의도: 외부화된 SKILL.md가 필요할 때만 로드되는 효율을 화살표 한 흐름으로 표현
